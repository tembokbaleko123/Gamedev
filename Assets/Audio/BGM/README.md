# BGM (Background Music)

Musik latar untuk game.

## Subfolder

### Battle/
Musik saat bertarung/combat.

**Karakter musik:**
- Fast tempo (120-140 BPM)
- Energetic
- Loop seamlessly

**File yang direkomendasikan:**
- `Battle_Theme_01.mp3` - Combat biasa
- `Battle_Boss_01.mp3` - Boss fight (jika ada)

### Menu/
Musik untuk main menu, pause menu, dll.

**Karakter musik:**
- Slow tempo
- Calm/ambient
- Loop seamlessly

**File yang direkomendasikan:**
- `Menu_Main.mp3`
- `Menu_Pause.mp3`

## Format File

| Format | Rekomendasi |
|--------|-------------|
| .mp3   | ⭐ Recommended (compressed, small size) |
| .ogg   | Alternative |
| .wav   | Jangan pakai untuk BGM (terlalu besar) |

## Unity Import Settings

- Load In Background: true
- Preload Audio Data: false
- Compression Format: Vorbis
- Quality: 50-80%

## Sumber Gratis

- https://freemusicarchive.org
- https://musopen.org
- Unity Asset Store (search "Free Music")
- YouTube Audio Library

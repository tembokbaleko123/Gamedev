# Swing SFX

Suara saat pedang/senjata diayunkan.

## File yang Direkomendasikan

| Nama File | Keterangan | Durasi |
|-----------|------------|--------|
| `Sword_Swing_Light_01.wav` | Ayunan cepat (light attack) | 0.2-0.4s |
| `Sword_Swing_Light_02.wav` | Variasi ayunan cepat | 0.2-0.4s |
| `Sword_Swing_Heavy_01.wav` | Ayunan lambat (heavy attack) | 0.4-0.8s |
| `Sword_Swing_Combo_01.wav` | Ayunan combo | 0.3-0.6s |

## Keyword Pencarian

- "sword swing"
- "weapon whoosh"
- "blade swing"
- "swoosh"
- "katana slash"

## Cara Pakai

Swing SFX dipanggil saat animasi attack dimulai, biasanya via Animation Event.

## Catatan

- Swing harus dimainkan SEBELUM hit impact
- Timing penting: swing → hit (jeda kecil) → impact
